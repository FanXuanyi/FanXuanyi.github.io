hexo+github搭建个人博客

blog

一直想搭建自己的博客，但碍于自己的技术有限，HTML、服务器部署、域名解析等等一大堆东西需要折腾。而Hexo是时下比较流行的博客框架，有比较丰富的主题和插件支持，Github为用户提供了GitHub Pages，空间免费稳定，用来搭建一个博客是非常合适的。

一、创建Github域名和空间

1、注册

注册一个Github帐号，注意username，这会影响你的域名，你的域名将会是username.github.io。

2、创建仓库

点击New respository按钮创建仓库，Repository name为username.github.io。

二、安装

1、安装git
2、安装Node.js
3、安装Hexo

三、编写、发布

1、创建博客
2、更改配置

进入刚刚生成的Hexo目录，安装主题。Hexo的主题

这里我选择了其中一个主题。

然后，进行基础配置。打开username.github.io目录下的_config.yml文件（详细的文件设置），修改几个键值对。

在username.github.io/source/_posts下写文章，例如hello-world.md文件。

在命令行中输入hexo s，测试服务启动，在浏览器中输入https://localhost:4000就可以访问了。

接下来，安装hexo-deployer-git自动部署发布工具。

3、发布

在浏览器中输入https://username.github.io就可以访问了。
